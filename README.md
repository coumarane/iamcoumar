# Hugo Blog

This repository contains the source code for my personal blog, built with [Hugo](https://gohugo.io/) — a fast and modern static site generator.  
It includes my articles on software architecture, DevOps, cloud technologies, and personal reflections.

## ✨ Features
- ⚡ Built with Hugo for fast static site generation
- 📝 Markdown-based content authoring
- 🌐 SEO-ready with sitemap and metadata
- 🎨 Customizable Hugo theme

## 🛠️ Deployment
Automated deployment via GitHub Actions.

## 📚 Content Topics
- Terraform, Azure, Kubernetes, GitOps
- Software design, automation
- Personal projects, tech notes
